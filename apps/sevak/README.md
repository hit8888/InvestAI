# Sevak App

Sevak is an AI assistant and routing service for the Meaku Agent-Admin Dashboard.

## Structure

- `routing-server/` - WebSocket server that provides conversational AI assistance and routing guidance

## Future Components

- `client/` - Frontend client for Sevak (coming soon)

## Development

```bash
# Start routing server
pnpm dev

# Build routing server
pnpm build

# Start routing server (production)
pnpm start
```
